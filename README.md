# 📺 YouTube Clone

A pixel-perfect front-end clone of YouTube's homepage, built from scratch using HTML, CSS, and JavaScript — replicating the core UI/UX of one of the world's most visited websites.

🔗 **Live Demo:** [techasmita.github.io/Asmita-website](https://techasmita.github.io/Asmita-website/)

---

## ✨ Features

- **Responsive Navbar** — YouTube-style header with hamburger menu, logo, search bar with voice search icon, notification bell, and user avatar
- **Sidebar Navigation** — Home, Shorts, Subscriptions, History, Playlists, Your Videos, Watch Later, Liked Videos
- **Subscriptions Panel** — Displays subscribed channel thumbnails (BeastBoyShub, BLACKPINK, Mythpat, Netflix, and more)
- **Filter Chips** — Category pills for All, Gaming, Music, JavaScript, Data Structures, Rowan Atkinson, and more
- **Video Grid** — Thumbnail cards with video duration, channel avatar, title, view count, and upload date
- **Clickable Thumbnails** — Each card links to the actual YouTube video

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/techAsmita/Asmita-website.git

# Open in browser
cd Asmita-website
open index.html
```

No dependencies. No build step. Just open `index.html` in any browser.
